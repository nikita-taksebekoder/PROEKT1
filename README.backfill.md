# Backfill инструкции

Этот файл описывает, как настроить и запустить автоматический backfill отзывов через GitHub Actions.

Шаги для настройки:

1. Создайте репозиторий на GitHub и добавьте этот проект.
2. В Settings → Secrets and variables → Actions добавьте секреты:
   - `ADMIN_KEY` — admin ключ, который воркер ожидает в заголовке `X-Admin-Key`.
   - `YCLIENTS_PARTNER_TOKEN` — (если используется) token партнёра YCLIENTS.
   - `YCLIENTS_USER_TOKEN` — (опционально) user token.

3. Опционально: в workflow используется `backfill.ps1` если он присутствует в репозитории. Если у вас есть готовый `backfill.ps1`, поместите его в корень репозитория. Workflow запустится по расписанию ежечасно и при ручном запуске.

4. Для тестового запуска можно запустить workflow вручную: Actions → Backfill YCLIENTS reviews → Run workflow.

Примечание по безопасности:
- Держите `ADMIN_KEY` в секрете; workflow использует его для авторизации импорта.
- Перед массовым импортом рекомендуем протестировать одну запись через `curl` и заголовок `X-Admin-Key`.
